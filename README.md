# Projet 4 : Modélisation et Machine Learning Supervisé

Ce projet consiste à mettre en place des modèles de machine learning supervisé dans le but d'effectuer des prédictions sur des individus.

## Description

Dans ce projet, j'ai développé des modèles de classification/régression (Machine Learning Supervisé) pour répondre à une problématique métier. Le projet est divisé en deux grandes parties : la création de nouvelles variables (Feature Engineering) et l'entraînement de différents modèles prédictifs.

## Contenu du dépôt

- `feature_enginering_04_2025.ipynb` : Notebook dédié à la préparation des données et à la création de features.
- `modelisation_04_2025.ipynb` : Notebook d'entraînement, d'optimisation et d'évaluation des différents modèles de Machine Learning.
- `requirements.txt` : Dépendances nécessaires à l'exécution des notebooks.

## Installation

1. Clonez ce dépôt :
   ```bash
   git clone https://github.com/KamelBenstaali/OpenClassroomP4.git
   ```
2. Installez les dépendances :
   ```bash
   pip install -r requirements.txt
   ```

## Compétences évaluées
- Feature Engineering
- Entraînement et optimisation de modèles de Machine Learning supervisé
